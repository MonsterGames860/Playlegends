World Manipulation

setBlock(x, y, z, blockId)

Places a single block of type blockId at the given world coordinates.

  • x, y, znumber — world coordinates
  • blockIdnumber — see Block IDs Reference
// Place a stone block at (10, 64, 10)
setBlock(10, 64, 10, 4); // 4 = Stone
fillRegion(x1, y1, z1, x2, y2, z2, blockId)

Fills the cuboid region between two corner points with blockId.

  • x1,y1,z1number — first corner
  • x2,y2,z2number — opposite corner
  • blockIdnumber — see Block IDs Reference
// Fill a 5x1x5 platform with planks
fillRegion(0, 70, 0, 4, 70, 4, 1); // 1 = Planks

Player & Entities

teleportPlayer(x, y, z)

Instantly teleports the local player to the given coordinates.

  • x, y, znumber — destination coordinates
// Teleport player to spawn
teleportPlayer(0, 80, 0);
spawnEntity(type, x, y, z)

Spawns an entity of the given type at the specified coordinates.

  • typestring — e.g. "sheep", "cow", "pig"
  • x, y, znumber — spawn coordinates
// Spawn a sheep near the player
spawnEntity("sheep", 5, 70, 5);

Chat & System

sendSystemMessage(msg)

Broadcasts a system-styled message to the in-game chat.

  • msgstring — message text
// Announce to all players
sendSystemMessage("The gates have opened!");
on(eventName, handler)

Registers an event listener. The handler runs every time the event fires.

  • eventNamestring — e.g. "onTick", "onPlayerChat"
  • handlerfunction — receives event arguments
// Listen for chat messages
on("onPlayerChat", function(name, msg) {
  console.log(name + ": " + msg);
});
registerCommand(name, handler)

Registers a chat command. The handler runs whenever a player uses it.

  • namestring — e.g. "heal"
  • handlerfunction — receives command args
// Register /heal
registerCommand("heal", function(args) {
  console.log("Healed!");
});
playSound(soundId, x, y, z)

Plays an uploaded sound at the given world coordinates.

  • soundIdstring — the uploaded sound file name
  • x, y, znumber — world coordinates
// Play a door sound
playSound("door_open.ogg", 0, 64, 0);

Voxel & World Interactions

on("onBlockPlaced", handler(x, y, z, blockId))

Fires whenever a player places a block. Useful for claim systems, custom structures, or portal mechanics.

  • x, y, znumber — coordinates of the placed block
  • blockIdnumber — see Block IDs Reference
// Detect a placed portal frame block
on("onBlockPlaced", function(x, y, z, blockId) {
  if (blockId === 89) {
    console.log("Portal frame placed at " + x + "," + y + "," + z);
  }
});
setBlockSilent(x, y, z, blockId)

Changes a block in the background with no placement particles or sound — useful for scripted world changes that shouldn't draw attention.

  • x, y, znumber — world coordinates
  • blockIdnumber — see Block IDs Reference
// Silently swap a trapdoor for stone
setBlockSilent(12, 64, 8, 4);
getBlocksInRadius(x, y, z, radius)

Returns a list of blocks (coordinates + block ID) found within radius of the given point.

  • x, y, znumber — center coordinates
  • radiusnumber — search radius, in blocks
// Scan a small area around the player
var nearby = getBlocksInRadius(0, 64, 0, 5);
console.log("Found " + nearby.length + " blocks");

UI & UX

createCustomHUD(config)

Adds a custom on-screen HUD indicator or bar (e.g. mana, stamina, thirst) driven by your script.

  • configobject — e.g. {"{ id, label, value, max, color }"}
// Add a simple mana bar
createCustomHUD({
  id: "mana_bar",
  label: "Mana",
  value: 80,
  max: 100,
  color: "#6fb3ff"
});
showNotification(title, message, icon)

Displays an in-game notification popup to the local player.

  • titlestring — notification title
  • messagestring — notification body text
  • iconstring — optional icon id / texture name
// Notify the player of a quest update
showNotification("Quest Updated", "Return to the village elder.", "quest_icon");
openCustomGUI(guiId)

Opens a custom interface for the player, such as a special inventory, a trader/shop screen, or a chest-style container.

  • guiIdstring — id of the registered custom GUI
// Open a trader screen
openCustomGUI("village_trader");

Entities & NPCs

spawnMob(mobType, x, y, z, stats)

Spawns a mob or NPC of the given type at the specified coordinates, with optional custom stats.

  • mobTypestring — e.g. "zombie", "npc_villager"
  • x, y, znumber — spawn coordinates
  • statsobject — optional, e.g. {"{ health, damage, speed }"}
// Spawn a tougher zombie guard
spawnMob("zombie", 10, 64, 10, { health: 40, damage: 6 });
on("onEntityDamage", handler(target, source, damage))

Fires whenever an entity takes damage, giving you the target, the damage source, and the amount dealt.

  • targetentity — the entity that was damaged
  • sourceentity — the entity or cause that dealt damage
  • damagenumber — amount of damage dealt
// Log every hit on a boss NPC
on("onEntityDamage", function(target, source, damage) {
  console.log(target + " took " + damage + " damage");
});

Audio & Atmosphere

setSkybox(textureId, timeOfDay)

Sets the sky texture, time-of-day cycle, and/or fog intensity for the current world.

  • textureIdstring — id of the uploaded skybox texture
  • timeOfDaynumber — 0–24000 tick value, or a fixed time
// Switch to a stormy dusk skybox
setSkybox("storm_sky", 13000);
playLoopingSound(soundId, x, y, z)

Plays a looping, positional ambient sound anchored at the given world coordinates.

  • soundIdstring — the uploaded sound file name
  • x, y, znumber — world coordinates
// Loop a campfire crackle at a fire pit
playLoopingSound("campfire_loop.ogg", 4, 64, 4);
